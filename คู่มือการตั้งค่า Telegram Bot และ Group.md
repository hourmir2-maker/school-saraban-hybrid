# 📖 คู่มือการตั้งค่า Telegram Bot และระบบกลุ่มประชาสัมพันธ์กลาง
สำหรับระบบ **School Saraban Hybrid**

คู่มือนี้จะอธิบายขั้นตอนการสร้างบอทส่วนตัวของโรงเรียน การตั้งกลุ่ม Telegram การดึงเลขรหัสกลุ่ม (**Group Chat ID** ที่ขึ้นต้นด้วย `-100`) และการผูกระบบแจ้งเตือนเพื่อใช้งานร่วมกันครับ

---

## 🤖 ส่วนที่ 1: วิธีสร้าง Telegram Bot และรับ Bot Token

หากโรงเรียนยังไม่มีบอทส่วนตัว ให้ดำเนินการสร้างบอทผ่านทาง **BotFather** ของ Telegram ดังนี้:

1. เปิดแอป Telegram พิมพ์ค้นหาผู้ใช้ชื่อ **`@BotFather`** (ตรวจสอบให้มีสัญลักษณ์ติ๊กถูกสีฟ้าข้างชื่อ)
2. กดปุ่ม **Start** หรือพิมพ์คำสั่ง **`/newbot`** เพื่อสร้างบอทใหม่
3. **กำหนดชื่อแสดงของบอท (Display Name)**: พิมพ์ส่งชื่อที่ต้องการ เช่น `น้องชบาสารบรรณ โรงเรียนบ้านควน`
4. **กำหนดชื่อ Username ของบอท**: พิมพ์ส่งชื่อภาษาอังกฤษ **ที่ต้องลงท้ายด้วยคำว่า `bot` เสมอ** (ห้ามมีเว้นวรรคหรือขีดกลาง เช่น `BanKuanChabaBot` หรือ `bk_saraban_bot`)
5. เมื่อตั้งชื่อผ่าน ระบบจะส่งข้อความแจ้งความสำเร็จมาให้:
   * **Token บอท**: ตัวเลขยาวๆ ที่มีเครื่องหมายโคลอน (เช่น `8927237989:AAHG8QIPFYN130pCakxzz7Kz2g-SBF4fcqw`) ให้คัดลอกเก็บไว้
   * **Username บอท**: พิมพ์เก็บไว้โดยไม่ต้องใส่เครื่องหมาย `@` (เช่น `BanKuanChabaBot`)

## 👥 ส่วนที่ 2: วิธีตั้งกลุ่ม Telegram Group เพื่อรับการแจ้งเตือนส่วนกลาง

1. **สร้างกลุ่มใหม่**: ในแอป Telegram ให้กดที่เมนูเลือก **New Group (กลุ่มใหม่)** -> เชิญคุณครูเข้าร่วมกลุ่มอย่างน้อย 1 คน -> ตั้งชื่อกลุ่มตามชอบ (เช่น `สารบรรณกลาง - โรงเรียนบ้านควน`)
2. **ดึงบอทเข้ากลุ่ม**: กดที่ชื่อกลุ่ม -> เลือก **Add Members (เพิ่มสมาชิก)** -> พิมพ์ Username บอทของคุณครูที่เพิ่งสร้างในส่วนที่ 1 ดึงเข้าร่วมกลุ่ม
3. **ตั้งบอทเป็นผู้ดูแลกลุ่ม (Admin) ⚠️ สำคัญมาก**:
   * เข้าไปที่หน้าข้อมูลกลุ่ม -> เลือกเมนู **Administrators (ผู้ดูแล)**
   * กด **Add Admin** -> เลือกบอทของโรงเรียน
   * เปิดสิทธิ์ **"Send Messages" (ส่งข้อความ)** ให้เรียบร้อย -> กดติ๊กถูกเพื่อยืนยันบันทึกสิทธิ์

## 🔍 ส่วนที่ 3: วิธีหารหัสกลุ่ม (Group Chat ID) ที่ขึ้นต้นด้วย `-100`

ความปลอดภัยของ Telegram จะใช้รหัสกลุ่มเป็นตัวเลขติดลบยาวๆ ขึ้นต้นด้วย `-100` (เช่น `-1002030405060`) วิธีดึงเลขนี้ที่ง่ายและรวดเร็วที่สุด มี 2 วิธีเลือกใช้ตามสะดวกดังนี้ครับ:

### วิธีที่ 1: ใช้บอทตัวช่วยหาให้ (วิธีที่แนะนำ - ง่ายและเร็วที่สุด)
1. ในกลุ่ม Telegram ของคุณครู ให้กดเพิ่มสมาชิก (Add Member) ค้นหาบอทตัวช่วยหาไอดีชื่อ **`@RawDataBot`** แล้วดึงเข้าร่วมกลุ่มชั่วคราว
2. ทันทีที่บอทเข้าร่วมกลุ่ม มันจะส่งข้อความโครงสร้างรหัสดิบ (JSON) มาให้ในกลุ่มยาวๆ
3. ให้มองหาบรรทัดที่ระบุว่า **`"chat": {`** แล้วดูค่า **`"id": -100xxxxxxxxx`** ด้านล่าง (ตามรูปตัวอย่างด้านล่าง)
4. คัดลอกตัวเลขทั้งหมด **รวมถึงเครื่องหมายลบด้านหน้า** (เช่น `-1002030405060`)
5. เมื่อได้รหัสแล้ว ให้ลบ (Remove) บอท `@RawDataBot` ออกจากกลุ่มได้ทันทีครับ

> **ตัวอย่างข้อความที่บอท @RawDataBot ส่งมา:**
> ```json
> {
>   "message": {
>     "chat": {
>       "id": -1002030405060,  <-- คัดลอกเลขชุดนี้ทั้งหมด
>       "title": "กลุ่มสารบรรณกลาง",
>       "type": "supergroup"
>     }
>   }
> }
> ```

---

### วิธีที่ 2: ดึงผ่าน URL เว็บบราวเซอร์ (กรณีไม่ต้องการดึงบอทภายนอกเข้ากลุ่ม)
1. ดึงบอทโรงเรียนเข้ากลุ่มและตั้งสิทธิ์ Admin ให้เรียบร้อย
2. พิมพ์ข้อความอะไรก็ได้ในกลุ่ม เช่น **`สวัสดีบอท`**
3. เปิดเว็บบราวเซอร์ เช่น Chrome หรือ Edge พิมพ์ลิงก์ขอประวัติแชทของบอทดังนี้ (แทนที่ `<BOT_TOKEN>` ด้วย Token ที่ได้จากข้อ 1):
   `https://api.telegram.org/bot<BOT_TOKEN>/getUpdates`
   *(ตัวอย่าง: `https://api.telegram.org/bot8927237989:AAHG8QIPFYN130pCakxzz7Kz2g-SBF4fcqw/getUpdates`)*
4. ค้นหาคำว่า **`"chat"`** ในหน้าเว็บบราวเซอร์ที่แสดงรหัสข้อความ จะพบข้อมูลห้องแชทกลุ่ม เช่น:
   `"chat":{"id":-1002030405060,"title":"..."}` 
5. คัดลอกรหัสตัวเลขนั้นเก็บไว้ใช้งาน

## ⚙️ ส่วนที่ 4: การนำไปตั้งค่าบนแอปสารบรรณ

เมื่อได้ข้อมูลครบถ้วนแล้ว ให้ล็อกอินเข้าสู่หน้าเว็บแอปพลิเคชันสารบรรณโรงเรียน และทำตามขั้นตอนนี้:

### ขั้นตอนสำหรับแอดมิน (หน้าจอตั้งค่า Settings):
1. ไปที่เมนู **"ตั้งค่าระบบ" (Settings)** 
2. เลื่อนลงมาที่หัวข้อ **"การตั้งค่า Telegram Bot API"**
3. กรอกข้อมูลดังนี้:
   * **Telegram Bot Token**: วางรหัส Token ที่ได้จาก BotFather 
   * **Telegram Bot Username**: ใส่ชื่อ Username บอท (ไม่ต้องใส่ `@`)
   * **Telegram Group ID**: ใส่รหัส ID กลุ่มที่ดึงมา (เช่น `-1002030405060` ต้องมีเครื่องหมายลบด้วย)
   * **ลิงก์คำเชิญเข้าร่วมกลุ่ม Telegram**: นำลิงก์กลุ่มของโรงเรียน (สร้างจากหน้าชวนเพื่อนในกลุ่ม Telegram เช่น `https://t.me/+AbCdEfGh123`) มาวางลงช่องนี้
4. กดปุ่ม **"บันทึกข้อมูล"** ด้านล่างขวา ระบบจะบันทึกพร้อมผูก Webhook ให้กับ Telegram บอททันที

### ขั้นตอนสำหรับคุณครูทุกคน (หน้าจอข้อมูลส่วนตัว Profile):
1. ไปที่เมนู **"ข้อมูลส่วนตัว" (Profile)**
2. ดูที่คอลัมน์ด้านซ้ายมือหัวข้อ **"กลุ่ม Telegram ของโรงเรียน"**
3. กดปุ่ม **"เข้าร่วมกลุ่ม Telegram ตอนนี้"** ระบบจะเด้งไปที่หน้าจอแอปพลิเคชัน Telegram เพื่อให้คุณครูกดเข้าร่วมกลุ่มกลางของโรงเรียนได้ทันที สะดวกสบายและป้องกันไม่ให้ครูทุกคนตกข่าวสารการลงรับคำสั่งใหม่ๆ ครับ!
